# Agentic Tax Benchmark

**Author:** Muhammad Bilal  
**Email:** aslammbilal61@gmail.com

Empirical benchmark measuring latency and token overhead
in Agentic vs. Non-Agentic LLM architectures.

## Key Finding

- +90.2% latency increase
- 15.57× token overhead ratio

- `agentic_tax_paper.docx` — Full research paper
- `benchmark.py` — Benchmark script
- `nab_results.csv` / `arl_results.csv` — Raw data
- `benchmark_results.png` — Results chart
